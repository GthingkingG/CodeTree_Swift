# CodeTree_Swift

CodeTree 플랫폼 Swift 문제 풀이 학습.

## 폴더 구조
날짜별 폴더(`YYMMDD/`)에 그날 풀이한 문제들을 저장. 각 문제 폴더 안에 README + Swift 파일.

```
250319/, 250320/, 250321/, 250322/, 250330/, ...
└── <문제명>/
    ├── README.md
    └── <solution>.swift
```

## 실행
```bash
cd 250319/<문제명>
swift <solution>.swift
```
